Расширение для браузера, которое непрерывно записывает кольцевой буфер последних N секунд активной вкладки. Когда что-то пошло не так — нажмите иконку и сохраните нужный момент в WebM или MP4.

## Возможности

- Непрерывная запись кольцевого буфера (5–40 секунд)
- Предпросмотр видео перед сохранением
- Обрезка клипа по времени с помощью ползунков
- Сохранение в **WebM** (мгновенно) или **MP4** (конвертация через ffmpeg.wasm)
- Нативный диалог сохранения файла ОС
- Горячая клавиша для быстрого захвата
- Все данные хранятся локально — ничего не отправляется на сервер

## Установка в Chrome

> Требуется Chrome 88 или новее.

1. Скачайте или клонируйте репозиторий на компьютер:
   ```
   git clone hhttps://github.com/LazarevR/BugCapture_extension.git
   ```
   Либо нажмите **Code → Download ZIP** на странице репозитория и распакуйте архив.

2. Откройте Chrome и перейдите по адресу:
   ```
   chrome://extensions
   ```

3. Включите **Режим разработчика** — переключатель в правом верхнем углу страницы.

4. Нажмите кнопку **«Загрузить распакованное»** в левом верхнем углу.

5. В открывшемся диалоге выберите папку с расширением (ту, где находится файл `manifest.json`).

6. Расширение появится в списке и иконка BugCapture появится на панели инструментов.
   - Если иконка не видна — нажмите на иконку пазла (Extensions) рядом с адресной строкой и закрепите BugCapture.

7. Перейдите на любую вкладку и нажмите иконку BugCapture, чтобы начать запись.

## Установка в Microsoft Edge

> Требуется Edge 88 или новее. Edge поддерживает расширения Chrome нативно.

1. Скачайте или клонируйте репозиторий (см. шаг 1 инструкции для Chrome).

2. Откройте Edge и перейдите по адресу:
   ```
   edge://extensions
   ```

3. Включите **Режим разработчика** — переключатель в левом нижнем углу боковой панели.

4. Нажмите кнопку **«Загрузить распакованный объект»**.

5. Выберите папку с расширением (ту, где находится `manifest.json`).

6. Расширение появится в списке. Иконку можно закрепить на панели через меню расширений (иконка пазла).

7. При первом запуске Edge может показать предупреждение о расширении не из магазина — нажмите **«Оставить расширение»**.

## Установка в Яндекс Браузер

> Требуется Яндекс Браузер 21 или новее. Основан на Chromium — полностью совместим с расширениями Chrome.

1. Скачайте или клонируйте репозиторий (см. шаг 1 инструкции для Chrome).

2. Откройте Яндекс Браузер и перейдите по адресу:
   ```
   browser://extensions
   ```
   Либо: **Меню (≡) → Дополнения → Управление дополнениями**.

3. Включите **Режим разработчика** — переключатель в правом верхнем углу.

4. Нажмите кнопку **«Загрузить распакованное расширение»**.

5. Выберите папку с расширением (ту, где находится `manifest.json`).

6. Яндекс Браузер может показать предупреждение о расширении из стороннего источника — подтвердите установку.

7. Иконка BugCapture появится на панели. Если не видна — нажмите иконку расширений и закрепите.

## Установка в Opera

> Требуется Opera 55 или новее. Opera основана на Chromium и полностью совместима с расширениями Chrome MV3.

1. Скачайте или клонируйте репозиторий (см. шаг 1 инструкции для Chrome).

2. Откройте Opera и перейдите по адресу:
   ```
   opera://extensions
   ```
   Либо: **Меню Opera (O) → Расширения → Управление расширениями**.

3. Включите **Режим разработчика** — переключатель в правом верхнем углу.

4. Нажмите кнопку **«Загрузить распакованное расширение»**.

5. Выберите папку с расширением (ту, где находится `manifest.json`).

6. Иконка BugCapture появится на панели инструментов. Если не видна — нажмите иконку куба (Extensions) и закрепите расширение.

## Использование

1. **Начало записи** — нажмите иконку BugCapture на панели инструментов. Расширение начнёт непрерывно записывать кольцевой буфер текущей вкладки.

2. **Захват момента** — нажмите иконку повторно (или используйте горячую клавишу). Откроется окно предпросмотра с последними N секундами записи.

3. **Обрезка** — используйте ползунки «Начало» и «Конец» чтобы выбрать нужный фрагмент. Временная шкала отображает выделенный диапазон.

4. **Сохранение:**
   - **WebM** — мгновенное сохранение, открывается стандартный диалог ОС.
   - **MP4** — конвертация через ffmpeg.wasm (~31 МБ, загружается один раз). Занимает несколько секунд в зависимости от длины клипа. Можно свернуть окно предпросмотра — конвертация продолжится в фоне.

5. **«Записать заново»** — закрывает окно предпросмотра и сразу возобновляет запись нового буфера.

### Горячие клавиши в окне предпросмотра

| Клавиша | Действие |
|---------|----------|
| `Пробел` или `K` | Play / Pause |
| `←` / `→` | Перемотка на 2 секунды |
| `Esc` | Закрыть окно |

## Настройки

Откройте настройки через **правый клик на иконке → Параметры расширения**.

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| Длина буфера | Сколько последних секунд хранить в памяти (5–40 сек) | 15 сек |
| Формат файла | WebM (быстро) или MP4 (совместимость) | MP4 |
| Горячая клавиша | Комбинация для быстрого захвата без клика по иконке | не задана |

## Требования

- Браузер на основе Chromium (Chrome, Edge, Яндекс, Opera) версии 88+
- Разрешение на захват вкладки выдаётся автоматически при первом нажатии

## Приватность

Расширение не собирает и не передаёт никакие данные. Вся запись, конвертация и хранение происходят исключительно локально на вашем компьютере.
